# 💰 Banking System

A simple **Java-based Banking System** project simulating core banking operations.  
This project demonstrates **OOP concepts**, **file handling**, and **user management** in a console-based application.

---

## 🛠 Technologies & Tools
- **Language:** Java ☕  
- **Build Tool:** Maven 📦  
- **IDE:** IntelliJ IDEA / Eclipse 🖥️

---

## ⚙️ Features
- 🏦 **Create bank accounts**  
- 💵 **Deposit & Withdraw money**  
- 📄 **View balance & transaction history**  
- 👥 **Manage multiple users & accounts**
